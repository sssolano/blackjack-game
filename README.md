# Blackjack Game

A simple browser-based Blackjack game built with vanilla HTML, CSS, and JavaScript.

## How to Play

1. Click **START GAME** to deal two cards.
2. Click **NEW CARD** to draw another card.
3. Try to reach **21** without going over.
   - Sum < 21 → keep drawing or stop.
   - Sum = 21 → Blackjack! You win.
   - Sum > 21 → Bust! You're out.
